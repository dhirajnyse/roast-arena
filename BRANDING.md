# Roast Arena Branding

## Recommended Name

**Roast Arena**

Tagline:

**Enter funny. Leave famous.**

Short pitch:

Roast Arena is a fast browser party game where friends battle with jokes, anonymous answers, chaotic voting, and dramatic judge verdicts.

## Alternate Names

- Roast Royale
- Punchline Arena
- Roast Rumble
- Joke Court
- Banter Battle

My recommendation is still **Roast Arena**. It is clear, searchable, expandable, and easy for people to repeat after hearing it once.

## Brand Personality

Roast Arena should feel:

- Bright
- Fast
- Social
- Competitive
- Safe-chaotic
- Streamer-friendly
- Slightly dramatic

It should not feel mean, dark, edgy-for-edgy's-sake, or like a generic AI chat app.

## Visual Direction

Logo style:

- Bright 3D game-show energy
- Chunky lettering
- Warm flame colors balanced by cool cyan and electric green
- Arena spotlight or badge shape
- Playful, not aggressive

Primary colors:

- Flame Orange: `#ff7a1a`
- Punch Pink: `#ff3fb4`
- Neon Cyan: `#18d8ff`
- Lime Pop: `#b6ff3b`
- Ink Navy: `#101426`
- Cream White: `#fff7df`

UI colors should use multiple accents instead of becoming a single orange or purple theme.

## Typography Direction

Logo:

- Chunky display lettering
- Rounded but bold
- High contrast

App UI:

- Clean sans-serif
- Dense and readable
- Big numbers for timers and scores
- Compact labels inside game controls

## Voice And Copy

The product voice should be playful and punchy.

Examples:

- "Start Room"
- "Enter Arena"
- "Drop Your Roast"
- "Reveal the Damage"
- "Judge Verdict"
- "One More Round"

Avoid long tutorials and heavy explanations inside the app. The game should teach itself through the flow.

## GitHub Repo Setup Steps

1. Go to [GitHub New Repository](https://github.com/new).
2. Repository name: `roast-arena`.
3. Description: `A browser party comedy game with prompt battles, voting, and dramatic judge verdicts.`
4. Choose visibility:
   - Private while we are experimenting.
   - Public once we have a polished demo.
5. Do not add a README, license, or `.gitignore` on GitHub if you want to push this local folder as-is.
6. Create the repository.
7. In this local project folder, run:

```powershell
git init
git add .
git commit -m "Initial Roast Arena prototype"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/roast-arena.git
git push -u origin main
```

If Git is not installed on this PC, install it from [git-scm.com](https://git-scm.com/downloads), restart PowerShell, then run the commands again.
